# Job Description Summary

Source: pasted Appian Senior Consultant job description.

## Role

Senior Consultant, Customer Success.

The role is post-sales and customer-facing. The consultant works throughout the project lifecycle to define, design, develop, and implement custom software and AI solutions on the Appian platform for commercial clients.

## Key Responsibilities

- Engage customers post-sales.
- Develop software solutions on Appian.
- Understand client business processes and technical needs.
- Launch relational data models in production.
- Develop APIs and integrations across multiple systems.
- Brainstorm and implement AI use cases.
- Apply RAG, agentic workflows, prompt engineering, and generative models.
- Collaborate with business users and technical teams.
- Drive adoption and empower clients to become self-sufficient.

## Requirements

- Mentoring/developing others and leading small teams.
- Business process definition and functional/technical requirements gathering.
- Communication, curiosity, problem solving, continuous learning.
- Cross-functional Agile teams, pair programming, standups, planning, sprints.
- Basic OOP understanding.
- Relational databases, database design, data modeling, SQL.
- Software testing practices.
- API integrations such as REST, SOAP, JDBC, web services.
- Familiarity with AWS, AI, analytics, machine learning, GCP, application integration, database, developer tools, governance, and containers preferred.
- 3+ years hands-on software development or technical consulting.
- B.S./B.A. in technical or related field.
- Travel 10% or less.

## Compensation

Base salary range: $100,000 to $150,000.

## Candidate Fit Themes

Strong fit:

- Booz Allen modernization: process modeling, requirements, transition planning, data mapping, API assumptions, workflow configuration, stakeholder validation.
- iCIMS SaaS integrations: APIs, SFTP, SAML, OAuth, enterprise implementation.
- Oracle technical consulting: Kubernetes, cloud-native deployments, observability, customer knowledge transfer.
- Resume skills: SQL, Python, BPMN/UML, AWS architecture, process/data modeling, Agile/Scrum.

Main gap:

- No direct Appian production development yet.

Best bridge:

> I have not built production Appian applications yet, but the consulting and implementation motion is very aligned with my background: model the process, understand the data, define integrations, validate acceptance criteria, and enable the customer.
